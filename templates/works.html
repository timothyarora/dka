Project 001: Three Phase Power Measurement with MAXQ3183 using SPI Communication

This project demonstrates the design and implementation of a complete three-phase power monitoring system using the Philips P89V51RD2 microcontroller and the MAXIM MAXQ3183 energy metering chip.

On the hardware side, I integrated precision voltage dividers and current transformers (CT1052) to accurately sense power parameters across all three phases. Communication between the microcontroller and energy metering chip was established using SPI, and output data was displayed on an LCD and transmitted to PC via RS-232.

On the software side, I developed embedded C code to configure the MAXQ3183 for continuous data acquisition and manage real-time display and logging. I also built a basic PC-side app for remote data monitoring and diagnostics.

Project 002: Battery-Integrated DC-DC Converter Design

I designed innovative battery-integrated DC-DC converters for low-voltage off-grid DC systems. I developed and validated advanced topologies like BBIC, BITIC, and QBIC, integrating battery management and high-efficiency features for solar homes, EVs, and portable systems.
